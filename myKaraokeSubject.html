Sujet du projet myKaraoke. Sup'Internet JavaScript deuxième année

Vous devez créé un karaoke sur navigateur. Le but est simple :
Votre page web elle, devra contenir un lecteur audio HTML5. Au fur et à mesure de la lecture du fichier audio, les paroles de la musique devront s'afficher sur votre navigateur.
Pour cela, vous devez utiliser les méthodes asynchrones vues en cours de JavaScript. Vous devez aller chercher un fichier de paroles (.txt) dans lequel les paroles d'une musique sont écrites.
Format du fichier de paroles à respecter : 
     -------------------------------------------

     [00:03.32]Wesh bébé tu m'anques trop tavu
     [00:07.25]J'aime trop quand t là serieux
     [00:12.02]Ouuuuhh yeah
     [00:19.55]...

     -------------------------------------------

Vous avez à votre disposition un fichier audio ainsi qu'un fichier de paroles (facultatifs, vous pouvez utiliser les votres si vous le désirez) :
  • http://supjs.fr/DEV2017/asynchrone/corrections/myKaraoke/music/LOR.mp3
  • http://supjs.fr/DEV2017/asynchrone/corrections/myKaraoke/lyrics.txt
